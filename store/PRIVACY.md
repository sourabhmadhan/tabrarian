# Privacy Policy

_Last updated: July 7, 2026_

This extension organizes your open browser tabs into named tab groups. It is
built to be private by design.

## Data collection: none

- The extension does **not** collect, transmit, sell, or share any data.
- It makes **no network requests**. All categorization and topic clustering
  runs locally in your browser. Favicons are served from Chrome's local
  favicon cache.
- Tab URLs and titles are read only in memory, only while the extension is
  organizing or previewing your tabs, and are never stored or sent anywhere.

## Data storage

The only data persisted is your settings (four flags: auto-group, focus mode,
reorganize-existing on/off, and grouping mode), stored via Chrome's
`storage.sync` so they follow your Chrome profile.

## Permissions

- `tabs` — read tab titles/URLs to decide which group each tab belongs to.
- `tabGroups` — create, name, and color Chrome tab groups.
- `storage` — save the four settings flags above.
- `favicon` — display tab icons in the popup from Chrome's local cache
  (instead of fetching them from the web).
- `alarms` — run the optional memory-saver sweep every few minutes to put
  long-unused tabs in collapsed groups to sleep, locally.
